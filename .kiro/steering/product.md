# 产品概述

边缘终端 — 基于 RuoYi-Vue 框架（v3.8.5）的安防监控管理平台。

## 用途
管理边缘设备（摄像头、门禁、传感器），处理告警记录、人员追踪和实时监控，面向物理安防场景部署。集成 AI 报告生成功能，通过 DeepSeek 兼容的 LLM API 实现。

## 核心功能
- 设备管理（摄像头、门禁、传感器、音频）
- 告警监控与记录管理
- 人员管理
- 实时数据大屏（DataV 可视化）
- 地图集成（Mapbox GL JS）用于空间监控
- MQTT 实时通信
- AI 报告生成（通过 OpenAI 兼容 API 的 SSE 流式输出）
- 任务调度与进程管理
- 系统管理（用户、角色、菜单、字典、配置）

## 部署目标
- Linux 边缘设备（主要目标）
- Windows 环境
- 通过 Nginx 反向代理，`/prod-api` 转发至 Java 后端

## 国际化
支持中文（zh，默认）和英文（en），通过 vue-i18n 实现，语言偏好存储在 cookie `language` 中。
